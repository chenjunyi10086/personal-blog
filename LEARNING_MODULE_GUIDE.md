### 📝 **默认笔记内容**

#### 1. Vue 3 响应式原理
- **核心概念**: Proxy 和 Reflect
- **依赖收集**: track() 函数
- **触发更新**: trigger() 函数
- **ref vs reactive**: 使用场景对比
- **代码示例**: 简版响应式实现

#### 2. JavaScript 异步编程
- **回调函数**: 最初的异步方案
- **Promise**: 链式调用，避免回调地狱
- **Async/Await**: 语法糖，同步风格异步代码
- **最佳实践**: 错误处理、并发请求

#### 3. CSS Flexbox 完全指南
- **容器属性**: display、flex-direction、justify-content
- **项目属性**: flex、align-self
- **常用布局**: 水平居中、三栏布局、卡片网格
- **实战案例**: 响应式卡片布局

#### 4. Node.js 事件循环
- **六个阶段**: Timers → Pending → Idle → Poll → Check → Close
- **setTimeout vs setImmediate**: 执行时机对比
- **process.nextTick**: 最高优先级微任务
- **微任务 vs 宏任务**: 执行顺序分析

#### 5. 常用排序算法
- **冒泡排序**: O(n²)，简单易懂
- **快速排序**: O(n log n)，常用高效算法
- **归并排序**: O(n log n)，稳定排序
- **选择建议**: 不同场景的最佳选择
